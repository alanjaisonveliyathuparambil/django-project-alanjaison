# SwiftScrap

A Django-based Scrap Pickup Management System.

## Features
- User Registration & Login
- Scrap Pickup Requests
- Scrap Price Management
- User Dashboard
- Collector Dashboard
- Admin Dashboard
- Help & FAQ System
- Notifications

## Technologies Used
- Python
- Django
- HTML
- CSS
- JavaScript
- SQLite
